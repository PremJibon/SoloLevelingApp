Read AGENT.md first and follow it strictly.

Design the core UI components for the "System" interface using NativeWind. Create:
- `SystemText`: A custom text component with a subtle blue outer glow.
- `SystemButton`: A sleek, semi-transparent button with a border that pulses when active.
- `SystemCard`: A container for quests and stats with a blurred background (using `expo-blur`) and a sharp cyan border.

All components should feel like a holographic interface floating in front of the user. Use `react-native-reanimated` for a subtle floating animation on the cards.
